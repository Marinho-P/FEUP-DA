{\footnotesize\ttfamily void Data\+::check\+Cities\+Water\+Deficit (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Checks and prints the cities with water deficit. This function calculates the water deficit for each city in the network by comparing the demand of each city with the actual water flow reaching it. It iterates over each city vertex in the graph and computes the flow of water reaching the city. If the flow is less than the city\textquotesingle{}s demand, it indicates a deficit. 

The complexity of this function depends on the number of cities and the number of incoming edges to each city vertex. Let n be the number of cities and m be the average number of incoming edges per city vertex. In the worst case scenario, it iterates over all city vertices and their incoming edges, resulting in a time complexity of O(n \texorpdfstring{$\ast$}{*} m).
